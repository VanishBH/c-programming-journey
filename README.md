# C Programming Journey

🚀 Learning and practicing C programming step by step

This repository contains structured C programming practice covering fundamental to advanced concepts.

## Topics Covered

### Core Concepts
- Loops
- Conditional Statements
- Functions
- Strings
- Structures
- File I/O

### Intermediate Concepts
- Pointers
- Recursion
- Bitwise Operations
- Preprocessor

### Advanced Concepts
- Dynamic Memory Allocation (malloc, calloc, realloc, free)
- Function Pointers
- Double Pointers
- Modular Programming (Multi-file programs)

## Purpose
To build strong problem-solving and programming skills in C.
